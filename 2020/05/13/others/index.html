对象检测

  • js实践
  • 2020-05-13 
typeof 最常用 对正则表达式使用typeof可能会反回‘function’
‘undefined’— undefined
‘boolean’ — 布尔值
‘string’ — 字符串
‘number’ — 数字
‘object’ — 对象或null
‘function’ — 函数
instanceof 判断是否属于某个对象的实例，两个对象必须在同一个全局作用域中
constructor 判断 someobj.constructor == Array/Object/Boolean…  //不是特别准确，因为该方法能被修改
Object.prototype.toString.call()，最为安全的方法，该方法不能检测非原生构造函数的函数名，开发人员定义的任何构造函数都将返回[object,Object],可以检测正则和原生对象
Object.prototype.toString.call(somejson) == “[object JSON]”;// 检测是否为原生json

typeof 可以准确的判断除object以外的基础数据类型，但不能区分object类型的具体类型，比如 Array 、Date,regExp 以及自定义类。

instanceof 的理解

虽然instanceof能够正确判断[] 是Array的实例对象，但不能辨别 [] 不是Object的实例对象
